# Networking

1. **IP Address**
    - The unique identifying number assigned to every device connected to internet
2. **Domain Name Serive (DNS)**
    - Internet phonebook, translating easy and friendly website name to machine readable IP Address

3. **Client**
    - A device which request information
4. **Server**
    - A computer that listens to client and response
5. **Network Protocols**
    - Standardized rules and formate that governs how devices exchange data
6. **Transmission Control Protocol (TCP)**
    - Data packets are delivered in the correct sequence
7. **User Datagram Protocol (UDP)**
    - a fast, connectionless internet protocol for time-sensitive applications like streaming, gaming, and DNS
8. **HyperText Transfer Protocol (HTTP)**
    - the fundamental system for transferring data between your client and a web server, using a request-response model to build the *World Wide Web*.
9. **WEBSOCKETS**
    - BiDirectional communication protocol
    - client or server can send data anytime

10. **Forward Proxy**
    - client ask data to FP and FP ask data to server
11. **Reverse Proxy**
    - RP filter clients request and forward to appropriate service
